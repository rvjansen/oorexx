Open Object Rexx - Files

Interpreter Package Source Files

This directory contains the ooRexx package manager and external library manager code. This code is responsible for loading and managing ::REQUIRES files and external library packages.

The following is a partial list of the files to be found in this subdirectory.

PackageManager.*
This the main package manager. This has methods for obtaining references to loaded ::REQUIRES files and also for resolving external routines and methods.
LibraryPackage.*
A class for managing the information in a loaded external library package.
