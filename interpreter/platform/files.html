Open Object Rexx - Files

Interpreter Platform Source Files

This is the main platform subdirectory for the platform-specific portion of the rexx shared library. Each major platform has a subdirectory devoted to it, and each subdirectory can have conditional compilations for a finer-grained tailoring of the environment.

The following are the available subdirectories.

windows The windows-specific platform adapter code.
unix The unix-specific platform adapter code. This covers all of the unix variants (AIX, Linux, Sun, Mac).
